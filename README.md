# Mighty Match - 3D Puzzle Game Landing Page

Responsive promotional landing page for **Mighty Match - 3D Puzzle Game**.

This project was created as a web landing page for a mobile puzzle game.  
The layout is based on a Figma design and includes adaptive versions for mobile and desktop screens.
A responsive Vite landing page for a mobile 3D puzzle game, built with HTML, CSS and JavaScript from a Figma design.

## 🔗 Live Demo

https://github.com/Mary1-com/STP-13024
stp-13024-phi.vercel.app

📌 Project Overview

The landing page presents the mobile game, its core features, advantages, gameplay experience, gallery, FAQ section, and legal pages.

The project includes:

responsive layout for mobile and desktop;
fixed header with navigation;
hero section with game statistics and Google Play call-to-action;
advantages section;
gallery section;
FAQ accordion;
footer navigation;
Privacy Policy page;
Terms and Conditions page;
hover, focus and pressed states for interactive elements.

🚀 Getting Started

To run the project locally:

npm install
npm run dev

To create a production build:

npm run build

To preview the production build locally:

npm run preview
📱 Responsive Design

The project is adapted for:

mobile layout: 375px
desktop layout: 1440px

The layout follows a mobile-first approach.

✨ Main Features
Header
fixed header;
navigation links;
responsive mobile menu;
hover, focus and pressed states.
Hero Section
game title and description;
rating/download/player statistics;
Google Play call-to-action button;
responsive background images.
FAQ Section
accordion behavior;
all items closed by default;
only one item can be opened at a time;
interaction without unwanted color changes.
Policy Pages

The project includes separate pages for:

Privacy Policy;
Terms and Conditions.

Each policy page includes a “Back to Home page” link and footer navigation.

🎨 Design Notes

The visual style is clean, colorful and game-oriented.
The landing page uses rounded elements, bright call-to-action colors, responsive images and interactive states according to the provided design requirements.
